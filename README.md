# Detección de Anomalías en Tráfico de Red con LSTM

Proyecto enfocado en detectar anomalías en tráfico de red utilizando redes neuronales LSTM y análisis de series temporales.

Los datos no están dentro del repositorio debido al tamaño de los datos. Puden consultarse en el siguiente enlace.
Documentación del dataset: https://research.unsw.edu.au/projects/unsw-nb15-dataset
